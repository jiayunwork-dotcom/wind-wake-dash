# 🌬️ 风电场尾流效应分析与发电功率预测系统

面向风电运营人员的 Streamlit 应用，用于分析风机之间的互相遮挡（尾流）影响，估算全场年发电量 (AEP)，并提供布局优化建议。

---

## ✨ 核心功能

### 1. ⚙️ 机型管理
- **内置 4 款常见机型**: DTU_2MW、Vestas_V112_3MW、SG_5MW_132、Haliade_X_8MW
- **自定义机型上传**: CSV 格式（功率曲线 + 推力系数曲线）
- 每款机型包含：额定功率、叶轮直径、轮毂高度、切入/额定/切出风速、完整功率曲线、推力系数曲线

### 2. 📍 风电场布局导入
- CSV 字段：`风机编号, X坐标, Y坐标, 风机型号`
- 内置 12 台风机示例布局（3×4 规则排布）
- **间距约束检查**：≥ 2倍叶轮直径可配置
- 交互俯视图：风机散点 + 叶轮圆圈，着色可视化

### 3. 🌬️ 尾流模型
- **Jensen (Park) 模型**: 线性锥形尾流，速度快，适合快速评估
  - 尾流扩展系数 α：陆上 0.075 / 海上 0.04 / 自定义
  - 公式：`δV/V₀ = (1-√(1-Ct)) / (1+αx/r₀)²`

- **Bastankhah 高斯模型**: 高斯截面分布，精度更高
  - 连续的尾流边缘速度恢复梯度
  - 受湍流强度 TI 影响的尾流扩展

- **多尾流叠加策略**:
  - 线性叠加 `Σ δV/V₀`
  - 均方根叠加 `√(Σ (δV/V₀)²)`

### 4. 🌬️ 风资源数据
支持三种输入方式：
1. **Weibull 参数生成**：形状因子 k + 尺度因子 A
2. **风向风速联合频率表 CSV**：(12/36 扇区 × 1m/s 分箱)
3. **各扇区 Weibull 参数 CSV**：每扇区独立 k、A、权重

输出：风向频率玫瑰图、平均风速、主导风向识别

### 5. 🧭 单风向工况分析
- 指定任意风向 + 风速
- **可视化**：
  - 俯视图叠加尾流锥形半透明区域
  - 风机按尾流损失着色（绿 → 黄 → 红）
  - 单机功率 / 入流风速表格
  - 尾流损失矩阵热力图（受影响 × 施加影响）

### 6. 📈 AEP 年发电量估算
- **逐风向 × 逐风速** 频率加权积分：
  ```
  AEP = 8760 × Σ P(dir, v) × f(dir, v)
  ```
- **关键指标**：AEP、无尾流 AEP、尾流损失电量、容量系数 CF
- **可视化**：
  - 风向-功率极坐标玫瑰图（0~359° 逐度扫描）
  - 标注最优风向（功率最高）+ 最差风向（尾流最严重）
  - AEP 各风向贡献饼图
  - 各风机尾流损失柱状图（Top5 红色高亮）
  - 综合尾流损失矩阵热力图

### 7. 🧬 布局优化（遗传算法）
- **目标函数**：最大化 AEP（含间距约束惩罚）
- 可配置：种群规模、进化代数、交叉/变异概率、变异步长、随机种子
- **每台风机独立设置**：是否允许移动 + X/Y 移动边界
- **实时进度**：进度条 + 每代统计信息
- **优化结果**：
  - AEP 提升百分比
  - 收敛曲线（历代最佳 + 种群平均）
  - 优化前后布局对比图
  - 推荐坐标 CSV 下载

## 🚀 快速开始

### 1. 安装依赖
```bash
pip install -r requirements.txt
```

### 2. 启动应用
```bash
streamlit run app.py
```
默认地址：http://localhost:8501

### 3. 一键体验
1. 打开「📍 风电场布局」→ 点击 **加载示例风场**
2. 打开「🌬️ 风资源数据」→ 选择 **生成示例数据** → **生成风资源数据**
3. 前往「📈 AEP与损失分析」→ 点击 **运行全场扫描计算**
4. 查看结果，或前往「🧬 布局优化」尝试改进布局

## 📐 关键业务约束

| 约束 | 实现 |
|------|------|
| 风机间距 ≥ 2D | `check_spacing_constraint()` 检查 + GA 惩罚项 |
| 切入/切出风速外功率=0 | `interpolate_power()` 边界返回 0 |
| Ct 额定风速后下降 | `build_thrust_curve()` 线性衰减段 |
| 气象风向(北0°顺时针)→数学坐标 | `_meteorological_to_math()` 转换 |
| 坐标旋转对齐风向 +X | `rotate_coords()` 统一矩阵运算 |
| numpy 向量化加速 | `combined_deficit_matrix()` 双层循环向量化友好 |
